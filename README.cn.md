# 中國年號查詢

[中國年號查詢](https://qinghaoh.github.io/chinese-era-explorer)是一個用於探索和學習中國歷史年號的交互式工具。這個項目旨在幫助用戶更好地了解中國豐富的歷史，通過直觀的界面和詳細的信息展示各個年號的起止時間和當時的君主。

## 功能

該工具支持三種搜索方式:

- 按年號搜索
- 按年份範圍搜索
- 按朝代/君主搜索

歡迎探索和學習中國歷史年號！
